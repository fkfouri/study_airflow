## Comandos
```bash
# informacoes
airflow info

# inicia um db
airflow db init

# Create User
airflow users create --username admin --firstname Firstname --lastname Lastname --role Admin --email admin@example.org --password password

# inicia o servidor em modo Detached
airflow webserver -D

# inicia o scheduler em modo Detached
airflow scheduler -D

```
